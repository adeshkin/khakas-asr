Мы добавили хакасский в Pontoon, платформу локализации Mozilla: https://pontoon.mozilla.org/kjh/common-voice/
Вам нужно будет довести вот эти пять строк до 100% перевода:
Если сомневаетесь в значении, посмотрите их переводы на русский: https://pontoon.mozilla.org/ru/common-voice/
После того, как эти строки будут переведены, в ближайшем релизе (они обычно случаются по понедельникам) откроется страница для сбора предложений: https://commonvoice.mozilla.org/kjh/write (сейчас там 404, сохраните ссылку на будущее). Вам нужно будет собрать 750 предложений, не защищённых авторским правом, которые носители вашего языка будут зачитывать вслух. Как только наберётся 750 предложений, хакасский будет доступен для сбора голосовых данных после ближайшего релиза!

Выложили на Hugging Face масштабный синтетический датасет для башкирского языка и ASR-модель, обученную на нем. 

🎙 Bashkort Voice (Синтетика)
Сгенерировали аудио с помощью OmniVoice. Использовали башкирские тексты, а как reference audio брали голоса из датасета sova_rudevices. 
Акцентируем внимание, что данные полностью синтетические, при этом общий объем составил 1891 час 0 мин 25 сек.
https://huggingface.co/datasets/AigizK/bashkort_voice

🤖 ASR-модель (w2v-bert-2.0)
Чтобы проверить синтетику в деле, обучили модель на объединении этого башкирского датасета и русского sova_rudevices. 
Результат: на тестовом сете Common Voice (башкирский язык) модель выдает WER = 0.1523. 
https://huggingface.co/AigizK/w2v-bert-2.0-bashkort-russian-omnivoice
